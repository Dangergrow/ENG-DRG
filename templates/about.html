LinguaMate

Умный ИИ-учитель английского языка от A0 до C2

ℹ️ О проекте

LinguaMate — это персональный ИИ-репетитор английского языка. Система использует искусственный интеллект (OpenRouter) для генерации уроков, проверки грамматики, разговорной практики и тестирования.

Возможности:
• 70+ уроков от A0 до C2
• Разговорный клуб с голосовым вводом и озвучкой
• Тестирование и сертификация уровней
• Обучающие игры (Виселица, Пары слов, Блиц-квиз, Анаграммы)
• Словарь с интервальным повторением
• Система достижений и XP
• Мульти-профили для всей семьи

Технологии:
• ИИ: OpenRouter (бесплатные модели)
• Озвучка: Microsoft Edge TTS (нейросетевые голоса)
• Распознавание речи: Web Speech API
• Бэкенд: Python + Flask
